# YuumiWiki Frontend

## 프로젝트 개요
개발 지식을 정리하는 개인 위키 서비스. 나중에 협업 기능 추가 예정.

## 마스코트
유미(Yuumi) - 파스텔톤의 귀여운 고양이

## 기술 스택
- Next.js 16 + React 19
- TypeScript
- Tailwind CSS
- **SSR (Server-Side Rendering)** - SEO와 초기 로딩 속도를 위해 서버에서 렌더링

## 도메인
- 프론트엔드: `yuumi.wiki`
- 백엔드 API: `api.yuumi.wiki`
- 로컬 개발: `localhost:3000` (프론트) / `localhost:8080` (백엔드)

## API
백엔드: Spring Boot

### 문서 조회
```
GET /api/docs/{문서제목}
```

응답 예시:
```json
{
  "title": "C Sharp",
  "alias": ["c#", "csharp", "씨샵"],
  "body": "# C#\n\n마크다운 본문..."
}
```

### 색상 팔레트
| 요소 | 색상 | 용도 |
|------|------|------|
| 본문 배경 | `#FFFFFF` | 가독성을 위한 순백색 |
| 헤더 배경 | 아이보리 계열 | 부드러운 느낌 |
| 인포박스 배경 | `#F9F4ED` | 따뜻한 아이보리 |
| 인포박스 헤더 | `#7FB3B3` | 민트 블루 포인트 |
| 링크 색상 | `#D35400` | 유미 오렌지 (강조) |

## 문서 형식
- 본문은 마크다운
- 내부 링크: `[문서제목](/docs/문서제목)` 형식
